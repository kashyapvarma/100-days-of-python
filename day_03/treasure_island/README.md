# 🏝️ Treasure Island

A text-based adventure game built using Python as part of **Day 3** of the **100 Days of Python Bootcamp**.

## 📖 Description

Your mission is to find the ultimate treasure.

The game features multiple story paths, enemies, dragons, magical items, and different endings based on the choices made by the player.

Every decision changes the outcome, so choose wisely.

## 🎮 Features

- Multiple story paths
- Monster battle
- Dragon storyline
- Final boss battle
- Multiple endings
- User input using nested `if`, `elif`, and `else` statements

## 🛠️ Concepts Practiced

- Python `print()`
- `input()`
- Variables
- Nested `if` statements
- `elif`
- `else`
- Program flow
- User interaction


## 📚 What I Learned

- How nested conditionals work
- Designing branching game logic
- Breaking problems into smaller parts
- Debugging indentation errors
- Testing every possible path before finishing a project
