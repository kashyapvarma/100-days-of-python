## 💻 Project

### Tip Calculator

A simple Python program that:

- Takes the total bill amount.
- Takes the tip percentage.
- Takes the number of people splitting the bill.
- Calculates the total amount including the tip.
- Splits the bill equally among everyone.
- Rounds the final amount to the nearest rupee.
