# 🤖 Day 6 - Reeborg's World

This is Day 6 of my **100 Days of Python** journey.

Unlike the previous days, today's exercises were completed using **Reeborg's World**, so there is no Python code file uploaded for this day.

## 📌 What I Practiced

- Defining and calling functions
- `while` loops
- `if`, `elif`, and `else` statements
- Using conditions to control program flow
- Breaking larger problems into smaller reusable functions
- Solving problems without knowing the exact number of steps beforehand

## 🚧 Hurdle Challenges

I completed multiple hurdle challenges where the robot had to:

- Jump over fixed hurdles
- Stop when it reached the goal
- Handle hurdles placed at random positions
- Jump over walls of different heights

These challenges helped me understand the difference between using a `for` loop for a known number of repetitions and a `while` loop when the number of repetitions is unknown.

## 🧩 Maze Challenge

The final challenge was to guide the robot through a maze using conditional logic.

I used the **right-hand rule**, where the robot:

1. Checks if the path to the right is clear.
2. Turns right and moves if possible.
3. Otherwise, moves forward if the front is clear.
4. Turns left if both paths are blocked.
5. Continues until it reaches the goal.
