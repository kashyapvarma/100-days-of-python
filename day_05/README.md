# 🔐 Day 5 - Password Generator

This is my Day 5 project from **100 Days of Code: The Complete Python Pro Bootcamp** by Dr. Angela Yu.

## 📌 Project Overview

This program generates a random password based on the user's preferences.

The user can choose:
- The number of letters
- The number of numbers
- The number of symbols

The program then creates a secure password by randomly selecting characters and shuffling them before displaying the final result.

## 🚀 Features

- User-defined password length
- Random uppercase and lowercase letters
- Random numbers (0-9)
- Random symbols
- Password characters are shuffled for better randomness

## 🛠️ Concepts Practiced

- Variables
- Lists
- User Input
- `for` Loops
- `range()`
- `random.choice()`
- `random.shuffle()`
- String Manipulation (`join()`)
- Python Functions

## 💻 Example

```text
How many letters would you like? 4
How many numbers would you like? 2
How many symbols would you like? 2

Generated Password:
a8$Bd2!Q
```

## 📚 What I Learned

- How to iterate using `for` loops.
- Using `range()` to repeat tasks.
- Selecting random items from lists.
- Building and shuffling lists.
- Converting a list into a string using `join()`.
- Breaking a problem into smaller logical steps.
